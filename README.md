# Results for the quickbench benchmark (https://github.com/Siomek101/quickbench)
I'm using this repo to document benchmarks on different distros.
My rules:
1. Same hardware
2. No Desktop Environment (unless no other way, and i can't uninstall it)
3. Running using SSH
4. Fresh OS, only installed git, and packages to compile.

## Tests on my old machine:
- Name: Baleron
- CPU: Intel(R) Core(TM)2 Quad Q9400 @ 2.66GHz
- RAM: 2x2GB DDR2
- 240GB IDE Hard Drive
